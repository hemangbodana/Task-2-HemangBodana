# Task-2-HemangBodana
Task-2 Aim- Build a basic classification model using a small dataset.

The Technical Pipeline
Input: 
Loading the dataset (150 samples across 3 classes) and performing a random shuffle and train-test split (typically 80/20) to remove order bias.

Process:
1. Scaling: Utilizing StandardScaler to balance data.
2. Algorithm: Implementing the K-Nearest Neighbors (KNN) algorithm.
3. Tuning: Selecting an optimal "K" value to avoid underfitting or overfitting.
4. Scikit-Learn Workflow: Instantiating the model, fitting it to training data, and predicting on test data.

Output Validation: 

Evaluating performance beyond simple accuracy using a confusion matrix (True Positives, False Positives, False Negatives, True Negatives) and metrics like the F1 Score.
